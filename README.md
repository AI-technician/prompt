# 🤖 프롬프트 엔지니어링 가이드북

> 중학생도 이해하는 AI 활용법 - 밝고 친근한 학습형 웹사이트

---

## 📌 프로젝트 개요

두 개의 GetUpNote 자료를 통합하여 만든 **프롬프트 엔지니어링 교육용 정적 웹사이트**입니다.
AI(특히 제미나이/Gemini)를 효과적으로 활용하는 방법을 중학생도 이해할 수 있도록 쉽고 친근하게 설명합니다.

---

## ✅ 구현된 기능

### 📄 페이지 구성
- **히어로 섹션**: 제목, 애니메이션 채팅 버블, 통계 카운터
- **섹션 1 - 프롬프트란?**: 비유(설계도), 좋은/나쁜 프롬프트 비교
- **섹션 2 - 4대 요소**: 페르소나, 작업, 맥락, 형식 + 통합 예시
- **섹션 3 - 5가지 기술**: 탭 형식의 인터랙티브 학습 (자기소개, 역할부여, 질문의질문, 꼬리질문, 예시활용)
- **섹션 4 - 6대 원칙**: 카드형 원칙 설명 + 체크리스트
- **섹션 5 - 실습 과제**: 4가지 실습 (운동계획, 여행일정, 요리레시피, 공부계획)
- **아웃트로**: 핵심 정리 및 복습 유도

### ⚡ 인터랙션
- 📊 상단 독서 진행도 바 (스크롤 연동)
- 📱 모바일 햄버거 메뉴
- 🗂️ 5가지 기술 탭 전환
- ✅ 체크리스트 + 실시간 피드백 (진행률 표시)
- 🎞️ 스크롤 페이드업 애니메이션
- 🔢 히어로 통계 카운터 애니메이션
- 💬 채팅 버블 순차 등장 애니메이션
- 💡 빈칸(blank) 툴팁 (마우스 호버 시 힌트)
- 🔝 상단 이동 버튼 (스크롤 400px 이후 노출)
- 📍 네비게이션 활성 섹션 하이라이트
